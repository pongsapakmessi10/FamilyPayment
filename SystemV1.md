# System V1 - The Family Bank

## ภาพรวมระบบ (System Overview)
เว็บแอปพลิเคชันสำหรับจัดการการเงินในครอบครัว ช่วยในการบันทึกรายจ่ายกองกลาง, จัดการหนี้สินระหว่างกัน, และตั้งเป้าหมายการออมเงิน โดยมีระบบอัปเดตข้อมูลแบบเรียลไทม์

## ฟีเจอร์หลัก (Key Features)

### 1. แดชบอร์ด (Dashboard)
- **สรุปภาพรวม**: แสดงยอดเงินคงเหลือ (Total Balance), รายจ่ายรวมรายเดือน (Monthly Expenses), และเงินออมรวม (Total Savings)
- **กราฟแสดงผล**: กราฟวงกลมแสดงสัดส่วนค่าใช้จ่ายแบ่งตามหมวดหมู่ (อาหาร, เดินทาง, ฯลฯ)
- **ข้อมูลล่าสุด**: ดึงข้อมูลจริงจากฐานข้อมูล

### 2. ค่าใช้จ่ายกองกลาง (Central Pot / Expenses)
- **บันทึกรายจ่าย**: สามารถเพิ่มรายการค่าใช้จ่ายใหม่ โดยระบุรายละเอียด, จำนวนเงิน, หมวดหมู่, และคนจ่าย (Payer)
- **รายการย้อนหลัง**: แสดงรายการค่าใช้จ่ายล่าสุดพร้อมวันที่และเวลา
- **Real-time**: เมื่อมีการเพิ่มรายการใหม่ ข้อมูลจะเด้งขึ้นหน้าจอทันทีสำหรับผู้ใช้ทุกคน

### 3. ระบบจัดการหนี้สิน (Debt Manager)
- **บันทึกหนี้**: สร้างรายการหนี้สินระหว่างบุคคล (ใครติดเงินใคร)
- **ล้างหนี้ (Clear Debt)**: ปุ่มสำหรับกดเคลียร์หนี้เมื่อมีการชำระคืนแล้ว สถานะจะเปลี่ยนเป็น "Paid"
- **Real-time**: สถานะหนี้สินอัปเดตทันทีเมื่อมีการเปลี่ยนแปลง

### 4. กระปุกความฝัน (Dream Jar / Savings)
- **ตั้งเป้าหมาย**: สร้างกระปุกออมเงินใหม่ ระบุชื่อของที่อยากได้และราคาเป้าหมาย
- **เติมเงิน**: สามารถกดเติมเงินเข้ากระปุกได้เรื่อยๆ
- **Progress Bar**: แสดงแถบความคืบหน้าเป็นเปอร์เซ็นต์และสีสันสวยงาม

### 5. ระบบเปลี่ยนภาษา (Multi-language Support)
- **TH/EN**: รองรับการใช้งานทั้งภาษาไทยและภาษาอังกฤษ
- **Switcher**: ปุ่มสลับภาษาที่ Sidebar (Desktop) และ Bottom Nav (Mobile)
- **Persistence**: จำค่าภาษาที่เลือกไว้แม้จะปิดหน้าเว็บไปแล้ว

## เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend**: Next.js, Tailwind CSS, Framer Motion, Recharts
- **Backend**: Express.js, MongoDB, Mongoose
- **Real-time**: Socket.io
- **Language**: TypeScript (Frontend), JavaScript (Backend)
